# BlackJack
My first Java project, the card game Black Jack. 

# Prerequisites
1. Java 8 or later
 
# How to use
Download the repo, and run the following in your command line.
``` 
$ cd BlackJack
$ gradlew run
```
Input "Y" or "N" and interact with the game in the command line.
